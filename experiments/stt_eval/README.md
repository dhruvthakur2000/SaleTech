# STT Bilingual Evaluation (English + Hindi)

## Objective
Evaluate offline speech-to-text models for bilingual (English + Hindi + Hinglish)
sales conversations and select the best candidate for SaleTech.

## Scope
- Small dataset
- Offline models only
- Accuracy + latency focused
- No production integration

## Models Planned
- Whisper
- Faster-Whisper
- Vakyansh

## Outcome
Final recommendation for production STT model.





## Audio rules:
   * Format: .wav
   * Mono
   * Sample rate: 16kHz
   * Length: 5–10 seconds
   * Content: conversational (sales-style)


## Dataset size (STRICT):
- English: 3–4 files
- Hindi: 3–4 files
- Hinglish: 3–4 files
- Total: 10–12 audio files * MAX *
